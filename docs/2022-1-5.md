## 低代码

## CSS

- [2022年的 CSS](https://zhuanlan.zhihu.com/p/452468980)
<br>在即将过去的 2021 年，CSS 变化非常地大，其中新增了很多特性，比如 CSS 容器查询、CSS 父选择器、CSS 层叠控制规则、CSS 子网格等等。而且这些特性已经在个别，甚至是在大部分主流浏览器中实现了
