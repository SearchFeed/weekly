# 前端&AI技术双周刊-2026.01.29


## 前端技术
- [Stylelint 17.0，现代化CSS代码检查工具迎来重大更新](https://stylelint.io/)
  <br>这款拥有超 100 个内置规则的 CSS 代码检查工具迎来 17.0 版本，包含大量破坏性更新和功能调整，官方同时提供了专门的迁移指南，帮助开发者完成版本升级。

- [基于 Rust 和 WebAssembly 构建 docfind](https://code.visualstudio.com/blogs/2026/01/15/docfind)
  <br>这是 VS Code 团队带来的实际案例分析，讲解了如何通过 WebAssembly 为站点搭建全新的高性能搜索系统，该系统基于 docfind 引擎开发，5 万篇新闻文章的检索演示展现了其极致的响应速度。

- [jQuery 4.0正式发布，迎来20周年重要更新](https://blog.jquery.com/2026/01/17/jquery-4-0-0/)
  <br>距离首次发布时隔 20 年，这款实际使用量极高的库迎来 4.0 版本，核心更新包括迁移至 ES 模块以兼容现代构建工具。作为 20 周年的重磅版本，jQuery 4.0 除放弃 IE 10 及旧版本支持、迁移至 ES 模块外，还新增了 Trusted Types 支持，在焦点事件处理上对齐了 W3C 规范，提升了库的安全性和标准化程度。

- [Node.js v25.4.0，require(esm)标记为稳定特性](https://nodejs.org/en/blog/release/v25.4.0)
  <br>Node.js Current 版本 v25.4.0 正式发布，核心更新为将 require(esm) 特性标记为稳定版，解决了 ES 模块引入的兼容性问题，让开发者可以更放心地使用该特性。

- [SVG 滤镜的妙用](https://www.amitmerchant.com/svg-filters-are-just-amazing/)
  <br>作者分享了自己对 CSS filter 属性的实践经验，通过丰富的案例展示 SVG 滤镜如何实现各类炫酷且高性能的视觉效果。

- [Firefox 147 开发者版发布，新增多项核心特性](https://developer.mozilla.org/en-US/docs/Mozilla/Firefox/Releases/147)
  <br>该版本更新内容丰富，新增 CSS 锚点定位、Navigation API、CSS 模块脚本、视图过渡类型等功能支持，同时在所有 Apple Silicon 芯片的 Mac 设备上默认启用 WebGPU 特性。

- [全新语义化元素 <geolocation> 正式推出](https://developer.chrome.com/blog/geolocation-html-element?hl=zh-cn)
  <br>自 Chrome 144 版本起，开发者可使用 <geolocation> 元素请求用户地理位置数据，无需通过 JavaScript 触发授权弹窗。

- [Tuesday, January 13, 2026 Security Releases](https://nodejs.org/en/blog/vulnerability/december-2025-security-releases)
  <br>Node.js 2026 年 1 月安全更新发布，包括 Node.js v20.x、22.x、24.x、25.x 多个版本的安全更新，修复了共计 5 个安全漏洞。

- [阿里通义千问重磅开源 Qwen3-TTS](https://github.com/QwenLM/Qwen3-TTS)
  <br>阿里通义千问团队正式开源了 Qwen3-TTS 系列语音生成模型家族。该系列采用端到端架构，支持秒级音色克隆、自然语言音色设计以及实时流式输出，极大降低了实时应用的门槛。仅需 3 秒参考音频，即可实现高保真零样本音色复刻。克隆后的音色支持跨语种无缝迁移，中文音色可直接用于说英语、日语、韩语、德语、法语、俄语、西班牙语、葡萄牙语、意大利语等10种主流语言，同时保留原音色特征。更进一步，还能自然输出四川话、北京话等多种中文方言，口音、神韵高度还原，为多语言内容创作和地方化应用打开新想象空间。
