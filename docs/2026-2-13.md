  <br>Node.js 25.6.0 是当前版本的最新更新，包含多项重要改进和新功能。其中包括 `async_hooks` 模块新增的 `trackPromises` 选项，`net` 模块的 `setTOS` 和 `getTOS` 方法支持，以及 ESM 在嵌入器 API 中的初步支持。此外，`TextEncoder` 的性能通过 `simdutf` 得到了显著提升，`stream` 模块新增了 `bytes()` 方法，`test_runner` 也引入了 `env` 选项。此版本还更新了 Ada 到 v3.4.2，支持 Unicode 17，并修复了多个安全性和性能问题。这些改进进一步增强了 Node.js 在服务器端和工具链中的表现，为开发者提供了更强大的功能和更高的性能。

## 资源推荐
- [新的包管理工具 npmx](https://npmx.dev/about)
  <br>npmx 是一个专注于提升 npm 包注册表用户体验的现代浏览器工具。它提供了快速、直观的界面，支持暗黑模式、键盘导航、代码浏览等功能，并允许连接到其他注册表（如 JSR）。此外，npmx 还提供了更好的管理界面，方便用户通过浏览器管理包、团队和组织。值得注意的是，npmx 并非包管理器或注册表，而是专注于优化浏览体验的开源项目，由社区共同维护。对于前端开发者来说，这是一个值得尝试的工具，能够显著提升 npm 包的使用效率。
