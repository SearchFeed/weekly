## 前端技术
- [Chrome 147 元素范围视图转换](https://developer.chrome.com/blog/element-scoped-view-transitions?hl=zh_cn)
  <br>Chrome 147 引入了 `Element.startViewTransition()` API，允许在 DOM 子树上启动视图转换，突破此前只能通过 `document.startViewTransition()` 进行全局转换的限制。核心价值: 支持并发运行多个视图转换、嵌套视图转换，并解决 `position: fixed` 内容的图层问题。文章提到从 Chrome 140 测试版以来的关键改进: 范围根默认自行参与、自动使用嵌套视图转换组、溢出内容自动裁剪等。提供了两个演示案例（随机列表、巴塔哥尼亚之旅），适合前端开发者了解 Web 动画能力的最新进展。

- [htmlparser2 v12.0.0 发布](https://github.com/fb55/htmlparser2/releases/tag/v12.0.0)
  <br>htmlparser2 是 Node.js 生态中最流行的 HTML 解析器之一，v12.0.0 是一次重大版本更新，核心目标是让 HTML 解析完全符合 WHATWG 规范。主要变更包括: raw-text 标签处理改进（iframe/noembed 等不再解析为 HTML）、SVG/MathML 标签名大小写规范化、注释和 DOCTYPE 解析修复、以及隐式标签闭合逻辑优化。

- [CSS corner-shape 属性详解](https://www.smashingmagazine.com/2026/03/beyond-border-radius-css-corner-shape-property-ui/?utm_source=CSS-Weekly&utm_campaign=Issue-638&utm_medium=web)
  <br>文章深入探讨了 CSS 新属性 `corner-shape`，它允许开发者创建非圆形的边角（如斜角、凹槽等），突破了 `border-radius` 的限制，为 UI 设计提供了更丰富的原生 CSS 样式能力。

- [ESLint v10.1.0 发布](https://eslint.org/blog/2026/03/eslint-v10.1.0-released/)
  <br>ESLint v10.1.0 版本发布，主要修复了 v10.0.0 引入的配置系统兼容性 Bug，并优化了部分规则的检测逻辑，提升了工具的稳定性与性能。

- [Next.js 16.2 发布](https://nextjs.org/blog/next-16-2)
  <br>Next.js 16.2 版本发布，重点提升了 Turbopack 的稳定性与构建性能，修复了部分预渲染 (PPR) 和 Server Actions 中的关键问题，为开发者提供更流畅的构建体验。

- [TypeScript 6.0 正式发布](https://devblogs.microsoft.com/typescript/announcing-typescript-6-0/)
  <br>该版本是TypeScript基于JavaScript代码基的最后一个正式版本，作为过渡版本衔接即将推出的Go编写的高性能TypeScript 7.0；新增无this函数类型推断优化、#/前缀子路径导入支持等特性，新增ES2025、Temporal API、Map upsert方法等标准类型支持，同时更新多项默认配置、废弃老旧特性，帮助开发者平滑过渡到后续7.0版本。

- [Vite 8.0 稳定版本正式发布](https://vite.dev/blog/announcing-vite8)
  <br>Vite 8是自Vite 2以来最重大的架构更新，替换原有双打包器架构为统一的Rust原生Rolldown打包器，生产构建速度提升10-30倍且兼容现有插件生态，还新增内置开发者工具、tsconfig路径原生支持等特性，可显著降低前端项目构建耗时，大幅优化开发体验。

## AI 资讯
- [MiniMax 发布 M2.7 自我进化模型](https://minimaxi.com/news/minimax-m27-zh)
  <br>MiniMax 发布旗舰模型 M2.7，最大亮点是"模型自我进化"能力。核心突破: ①自我迭代 - M2.7 能构建复杂 Agent Harness，参与自身强化学习过程，在 MLE Bench Lite 获得 66.6% 得牌率；②软件工程能力 - SWE-Pro 得分 56.22% 追平 GPT-5.3-Codex，VIBE-Pro 55.6% 接近 Opus 4.6；③专业办公 - GDPval-AA ELO 1500 位列前四，Office 三件套编辑能力显著提升；④原生 Agent Teams - 支持多智能体协作，内化角色边界和对抗推理能力；⑤深度集成 OpenClaw - MM Claw 评测正确率 62.7%，接近 Sonnet 4.6。

- [小米发布 MiMo-V2 旗舰模型](https://mimo.xiaomi.com/mimo-v2-pro)
  <br>小米发布旗舰基础模型 MiMo-V2-Pro / Omni / Flash / TTS，专为代理工作负载设计。核心亮点: ①万亿参数规模 - 总参数超1T，活跃参数42B，混合注意力比例提升至7:1；②顶尖排名 - Artificial Analysis全球第8，中国第2；PinchBench全球第3，ClawEval全球第3（接近 Opus 4.6）；③编码能力 - 内测超越 Claude 4.6 Sonnet，接近 Opus 4.6；④代理优化 - 深度优化工具调用和多步推理，是 OpenClaw 的原生大脑；⑤高性价比 - 1M上下文API定价显著低于 Claude。

- [通义千问3.5-Max预览版发布](https://www.alibabacloud.com/blog/qwen3-5-max-preview-unveiled_602969)
  <br>阿里通义千问发布 Qwen3.5-Max-Preview 预览版，在 LM Arena 匿名评测中得分 1464，跻身全球前五、中国第一。关键数据: ①无风格控制排行榜全球第6，中国第1；②数学能力全球第5；③专家级文本能力全球第10。同时 Qwen3.5 系列已开源 8 个模型（0.8B-397B参数），其中 Qwen3.5-Plus 采用 MoE 架构（397B总参数/17B活跃参数），性能超越2-3倍参数量的竞品。这标志着国产大模型在全球排行榜上取得了重要突破。

- [阿里推出 Accio Work 商业AI代理平台](https://www.accio.com/work)
  <br>阿里巴巴推出全球首个面向真实业务的AI代理平台 Accio Work，定位为本地优先的桌面AI代理。核心能力: ①全流程自动化 - 涵盖设计、采购、店铺运营、营销、CRM等业务环节；②多代理协作 - Teams功能支持代理团队协同工作；③企业级安全 - 沙盒隔离、四层权限模型、本地优先架构；④丰富集成 - 支持浏览器自动化、连接器（Gmail/Slack/Notion）、渠道（Telegram/Discord/WeChat）。支持 Gemini、GPT-4o、Claude、Qwen 多模型切换，是 Claude Cowork 的有力竞品，适合电商和跨境卖家使用。

- [Claude Code 推出自动模式](https://claude.com/blog/auto-mode)
  <br>Anthropic 为 Claude Code 引入 auto mode 权限模式，作为 --dangerously-skip-permissions 的安全替代方案。核心机制: ①智能分类器 - 执行前检查每个工具调用，自动拦截破坏性操作（批量删除、数据泄露、恶意代码）；②安全与便利平衡 - 安全操作自动执行，风险操作被阻止并引导 Claude 换路径；③降级保护 - 持续触发阻止时才会请求用户确认。目前为研究预览版，建议在隔离环境中使用，对需要长时间运行的自动化任务是重要改进。

- [Claude 新增电脑使用能力](https://claude.com/blog/dispatch-and-computer-use)
  <br>Anthropic 为 Claude Cowork 和 Claude Code 推出 Computer Use 功能，让 Claude 能直接操作用户的电脑完成任务。核心特性: ①自动工具选择 - 优先使用连接器（Slack/Calendar），无连接器时则控制浏览器、鼠标、键盘；②Dispatch 远程任务 - 手机发指令，桌面端执行，支持定时任务；③安全机制 - 需显式权限、可随时中断、自动检测威胁。目前为研究预览版，仅支持 macOS，适合开发者探索 AI 自主操作电脑的可能性，对自动化工作流有重要意义。

- [GPT-5.4 mini与nano正式发布](https://openai.com/zh-Hans-CN/index/introducing-gpt-5-4-mini-and-nano/)
  <br>OpenAI本次推出的两款小型大模型，在代码能力、推理、多模态理解等维度较前代均有显著提升，运行速度最高超前代两倍，成本大幅降低，可广泛应用于代码开发、子智能体调度、实时多模态任务等场景，API及ChatGPT等多渠道已同步上线，性价比突出。
